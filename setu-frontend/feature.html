Security

Setu is designed to be private and secure

Your conversations and files stay protected with authentication and encrypted transit.

🔑

JWT Authentication

Token-based auth powered by Appwrite ensures only authorized users can create and join rooms. Tokens are verified on every API call and WebSocket connection.

🔒

Encrypted Transit

All messages and file transfers are sent using encrypted connections. The messages and files are locked and can only be opened by the person they are meant for.

🆔

Session Tracking

When you connect to our service you get a code that is just for you. Our server keeps track of who's connected to each room so we can show you who is online.

👤

User Profiles

If you log in to our service you get your profile where you can add a nickname, phone number and country. You can see your profile in the sidebar. You can even log out quickly from there.
